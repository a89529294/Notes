`background-image: conic-gradient(from 0deg at 50% 50%, color1, color2)` Note that first argument `from 0deg at 50% 50%` is the default value. 

```css
.sharp-conic {
	background-image: conic-gradient(
      deeppink 0%,
      deeppink 33.3%,
      gold 33.3%,
      gold 66.6%,
      slateblue 66.6%,
      slateblue 100%
    );
}
```
