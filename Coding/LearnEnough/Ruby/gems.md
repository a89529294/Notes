## Gemfile
- `gem "rails"` installs the latest gem version
- `gem "rails" ">= 3.26"` installs the latest gem version as long as it is greater than *3.26*
- `gem "rails" "~> 3.26"` installs the lastest version *LESS THAN*  `4.0`
