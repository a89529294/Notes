INNER JOIN SYNTAX: 
``` plsql 
SELECT table1.col_name1, table2.col_name2  
FROM table1 [AS a]  
[INNER] JOIN table2 [AS b]  
    ON table1.id = table2.id;
```

Types of Joins:
	1. Inner Join
	2. Outer Join
	3. Left/Right Koin
	4. Cross Join
	5. Union Join
